# Tools

## REQUIREMENTS

- Node.js Active LTS
- netcat
- ghq
- peco | fzf
- (Linux) xsel | (macOS) pbcopy

## Install

clone repository:


```
$ git clone https://github.com/KidsCannon/tools.git $HOME/.k8n/tools
```

add to your shell rc file. (.bashrc, .zshrc etc):

```
source $HOME/.k8n/tools/init.sh
```
